## Setup PHP

### php.ini
Über die php.ini-Datei können gewisse Einstellungen an der PHP-Instanz global angepasst werden. 
```ini
memory_limit   = 128MB
display_errors = off
error_logs     = /Applications/MAMP/logs/php_error.log
```
