## Setup PHP

### php.ini
Über die php.ini-Datei lässt sich PHP konfigurieren, so z.B. der maximale 
```bash
memory_limit   = 128MB
display_errors = off
error_logs     = /Applications/MAMP/logs/php_error.log
```
